ShieldAI

AI-Powered Sensitive Data Leakage Detection Platform

Overview

ShieldAI is a security-focused web application that helps identify and prevent accidental exposure of sensitive information. The platform scans user-provided content, detects confidential data, calculates risk levels, and generates a sanitized version of the content for secure sharing.

Features

* Email Detection
* Phone Number Detection
* Aadhaar Detection
* PAN Detection
* Password Detection
* Risk Score Calculation
* Risk Level Classification
* Sanitized Output Generation
* Real-Time Analysis

Tech Stack

Frontend

* React.js
* Vite
* CSS

Backend

* FastAPI
* Python
* Regular Expressions (Regex)

How It Works

1. User enters content containing sensitive information.
2. ShieldAI scans the content in real time.
3. Sensitive data is detected and categorized.
4. Risk Score and Risk Level are generated.
5. A sanitized version of the content is produced by masking confidential information.
